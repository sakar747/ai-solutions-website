# AI Solutions Website

Professional Django website for an AI software company with:

- Professional business pages
- AI virtual assistant
- Client inquiry form
- Feedback form
- Admin login
- Admin dashboard
- Inquiry management
- MongoDB Atlas support with local JSON fallback

## Local Setup
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
```

Open: http://127.0.0.1:8000/

Default local admin:

```text
Username: admin
Password: Admin@123
```

Use environment variables to change credentials.
